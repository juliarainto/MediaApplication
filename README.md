# Media Compressor

Client-side: HTML, CSS, JavaScript, jQuery

Server-side: Node.js, Express

Demo here: https://juliarainto.com/apps/mediapakkaaja


Start
> npm install -g http-server

Server:
> cd .\server
> npm install
> npm start

Client:
> cd .\client
> http-server
> open browser to: http://localhost:8080

Build:
> npm build
